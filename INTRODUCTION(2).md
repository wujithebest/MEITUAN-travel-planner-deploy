# AI 本地生活路线规划系统

## 1. 产品定位
基于 FastAPI + React 的 AI 路线规划系统，通过自然语言交互，结合天气、交通等多源数据，为用户生成个性化出行方案。

## 2. 核心功能
- **双模式规划**：自由探索（AI推荐）+ 精准规划（用户指定路径）
- **智能路线生成**：时间预算自动分配、POI多维度评分筛选
- **实时信息整合**：营业状态、评分、交通路况
- **地图可视化**：高德地图JS API 2.0，路线polyline绘制

## 4. AI Pipeline四步流程
1. **意图解析**：LLM理解用户需求，输出结构化JSON
2. **宏观规划**：区域定位、主题映射、天数分配
3. **微观填充**：POI搜索、路线优化、餐厅插入
4. **输出渲染**：结构化行程单 + 地图数据

### 4.1 意图解析示例
**输入**："想带家人去徐汇区玩两天，喜欢历史建筑，预算中等"
**输出**：
```json
{
  "plan_mode": "intent",
  "area": "徐汇区",
  "days": 2,
  "theme": "历史",
  "preferences": {"pace": "休闲", "budget": "中等"}
}
```

### 4.2 POI评分算法
```
POI综合得分 = 基础评分×2.0 + 评论数对数×1.5 - 距离惩罚×0.5 + 主题匹配加分 + 多样性奖励
```

## 5. 用户体验流程
1. **欢迎界面**：选择规划模式（自由探索/精准规划）
2. **需求输入**：自然语言描述出行需求
3. **AI处理**：流式输出规划进度
4. **结果展示**：结构化行程 + 地图可视化
5. **迭代优化**：支持追问和调整
